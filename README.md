# EEC172-Project

## Pinout

### Speaker
Pin 29 or 30 (PWM)

### RFID Sensor
Pins 1 (UART1 TX), 2 (UART1 RX), 61 (IRQ), 62 (RST)

### OLED
Pins 15 (OC), 18 (RST), 53 (DC), 8 (CS), 5 (CLK), 6 (MISO), 7 (MOSI)

### PIR Sensor
Pin 16 (Data)

### IR Receiver
Pin 3 (Data)

### Computer
Pins 55 (UART0 TX), 57 (UART0 RX)
