# EEC172-Project

## Pinout

### Speaker
Pin 17 (PWM)

### Optical Fingerprint Sensor
Pins 1 (UART1 TX), 2 (UART1 RX)

### OLED
Pins 15 (OC), 18 (RST), 53 (DC), 8 (CS), 5 (CLK), 6 (MISO), 7 (MOSI)

### PIR Sensor
Pin 62 (Data)

### IR Receiver
Pin 3 (Data)

### CC3200
Pins 4 (SW3), 64 (Red LED)

### Computer
Pins 55 (UART0 TX), 57 (UART0 RX)
